1. Login screen

The registered users can login with their assigned user IDs and passwords. New users cannot sign up here because only administrators can add users. User Control button provided on the bottom of the homepage

2. Main Menu

This is main page after login. Any user is logged out after 1 hour of inactivity. After login, user can always see logout button in the top right of any page in the system.

User must select a patient before clicking any button except for buttons in the bottom. User can type some part of patient's name, and then web page will list patient names matching characters entered by user. User can select any patient from the list. User can also search by MRN.

Clicking on the advanced search button displays a list of patients. You can filter patients based on MRN, Name, Workup status, LCEC Status, Primary MD, Referring MD and Date of referral. Type into the text boxes to filter the result displayed. Click on the “Select “ button to choose a particular patient.

After selecting any patient, you can see that patient name, MRN, work up status, and LCEC status filled automatically.

Patient Data

Patient Data page is displayed when user clicks Patient Data Button in Main Menu. If user want to make some changes of patient data, then user can modify any fields and click save change button in this patient data page. Also, if user selects any buttons in main menu, then navigation bar is activated. User can click any link user wants to go.

Lesion

This is Lesion page. Here, user can go schedule Testing page or Auto Schedule Testing Page using two buttons easily. User can make changes or add new Lesion, using two buttons. The window like below will popup for adding new Lesion.

Fill in the details in 'Patient Risk Factors ' tab. To add new lesion details click on the 'add new lesion' button.Fill in the details and click on the add lesion button to update the database.

To delete a lesion click on the corresponding 'delete lesion' button.Click on the save changes button to save the changes made.

Delete Patient

To delete a patient select a patient and then click on the delete patient button.It will promt the user for a confimation . Click 'yes' to confirm deletion and 'No' to cancel deletion.

Create New Patient Page

Clicking on the Create New Patient button which is at the bottom of the page lets you create a new patient. Enter the details required in the corresponding text box fields. Click on the Reset button to reset all the text box contents. Click on the Create patient button to update the database with the new patient details. Click on the Logout button to logout of the application.
